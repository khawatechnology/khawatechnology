Client:

  • Carrfour is a major retailer in France and in the world.
  • The technical stack is based on Ruby on Rails Solidus framework coupled with a React / Redux frontend. The website is hosted on Heroku.

Responsibilities:

  • E-commerce backend development with the Solidus framework.

Completed Tasks:

  • Implement data flows for automatic information sharing between the company's different entities: finance, marketing, customer satisfaction, etc.
  • Implement data flows with providers to synchronize products' data.
  • Implement custom Solidus modules for customer returns management, refunds, multi-package and multi-warehouse deliveries.
