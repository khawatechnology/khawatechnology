Client :

  • Le client est un acteur majeur de la grande distribution en France et dans le monde.
  • La stack technique est basée sur le framework Ruby on Rails Solidus couplé avec un frontend React / Redux. Le site est hébergé sur Heroku.

Responsabilités :

  • Développement backend de fonctionnalités e-commerce avec le framework Solidus

Tâches réalisées :

  • Implémentation de flux pour l'échange automatique d'informations entre différentes entités de l'entreprise : finance, marketing, satisfaction client, etc.
  • Implémentation de flux avec les fournisseurs pour synchroniser les données produit
  • Implémentation de modules Solidus personnalisés pour gérer les retours clients, les remboursements et les livraisons multi-colis depuis des entrepôts multiples.
